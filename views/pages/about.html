This website uses EJS, bootstrap, and node.js to create a dynamic site with headers and footers on each page

Each tab links to another page of the site, but every page has the same header and footer with tabs as links

The square tab brings the user to a page, where numbers can be entered. The result will be that number squared

The cube tab brings the user to a page, where numbers can be entered. The result will be that number cubed

The home tab brings the user back to the original page
